How to use WinZip Self Extractor

Before you start the WinZip Self Extractor

    Create a text file that contains the message you want to display when the exutable starts.
     Create a batch file that copies the exercises, and store it in the main folder for the files to be zipped.
     Create the zip file.

How to create an exutable file

    Run the WinZip Self Extractorand click through the first three dialogue boxes.
     Enter the name of the zip file in the fourt dialogue box
     Click the next button to test the exutable
