How to use WinZip Self Extractor

Before you start the WinZip Self Extractor

  1. Create a text file that contains the message you want to display when the exutable starts.
  2. Create a batch file that copies the exercises, and store it in the main folder for the files to be zipped.
  3. Create the zip file.

How to create an exutable file

  1. Run the WinZip Self Extractorand click through the first three dialogue boxes.
  2. Enter the name of the zip file in the fourt dialogue box
  3. Click the next button to test the exutable
